# creator-Path

CreatorPath – AI-Powered Content Creator 


Overview

CreatorPath is an interactive web application designed to help aspiring content creators build their personalized roadmap. Users provide their niche, content type, face-on-camera preference, tools, and budget, and CreatorPath generates a globally applicable, step-by-step content strategy. The roadmap includes AI-generated hooks, scripts, filming angles, and posting tips, making it easier for creators to produce impactful content.

Key Features

Dynamic Questionnaire: Capture user preferences to tailor the roadmap

AI-Powered Roadmap Generation: Personalized scripts, hooks, and tips

Visual Interactive Timeline: Roadmap milestones presented as cards and icons

Export & Save Options: PDF export and optional user-specific roadmap saving

Cozy & Aesthetic UI/UX: Modern, engaging design for today’s creators

Responsive Design: Optimized for both mobile and desktop

Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js / Express (or Firebase)

AI Integration: OpenAI API

Visualization: Dynamic timeline & cards
